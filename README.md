# 🎉 ByteFest 2K25 – Official Event Website

Welcome to the official repository for **ByteFest 2K25**, a tech fest organized by our college.

🌐 **Live Website:** [bytefest2k25.netlify.app](https://bytefest2k25.netlify.app)
